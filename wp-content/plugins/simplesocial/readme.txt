== Description ==

Display icons for your social media profile links.

Icons are set under *Appearance > Menus*.

* Simply create a custom menu, add the desired icon names to the respective Link Text/Navigation Labels (Facebook, Twitter, etc.), assign it to the SimpleSocial menu area, and use the shortcode `[simplesocial]` in any Page, Post, or Text Widget
* You can drag and drop icon menu items in any order you like (just like any normal WordPress menu)
* You can set the universal icon size and color under *Settings > SimpleSocial*

Icons by [Simple Icons](https://simpleicons.org/) | Plugin by [Bryan Hadaway](https://calmestghost.com/ "Web Designer")

Supported icons:

* Facebook
* Twitter
* LinkedIn
* YouTube
* Instagram
* Pinterest

1500+ more in the [pro version](https://simplesocial.pro/).
